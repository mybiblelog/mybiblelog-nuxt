# 어디서나 설치

My Bible Log는 웹사이트이면서 \"프로그레시브 웹 앱\"(PWA)이기도 해, 어떤 기기에도 독립 앱처럼 설치할 수 있습니다.

<div style="margin: 1rem">
  <img alt="My Bible Log 데스크톱 앱 스크린샷" src="/screenshots/sc8-install-anywhere.webp" />
</div>

예를 들어 Google Chrome으로 다음 환경에 PWA를 설치할 수 있습니다:

* Windows
* Mac OS
* Linux
* Android
* iOS

PWA 설치는 먼저 브라우저에서 사이트를 여세요. Chrome에서 설치하는 방법:

[Chrome에서 PWA 설치](https://support.google.com/chrome/answer/9658361)

## 키보드 없이 입력

이동 중에 읽거나 손에 기기가 있을 때 쓰기 쉽도록 설계되었습니다.

성경 구절은 타이핑 없이 선택할 수 있어 휴대폰·태블릿 등 터치 기기에서 편합니다.

## 최신 버전 유지

업데이트는 일반 웹사이트와 같습니다.

페이지를 새로고침하면 최신 버전을 쓰게 됩니다.
