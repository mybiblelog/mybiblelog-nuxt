# 노트 페이지

<div class="phone-frame">
  <img alt="My Bible Log 노트 페이지 스크린샷" src="/screenshots/sc10-notes.webp" />
</div>

읽으며 생각을 모으고 정리할 수 있도록 자유 형식 노트를 지원합니다.

노트 페이지에서는 새 노트를 만들고 기존 노트를 다룹니다. 노트가 많아지면 정리가 중요합니다.

사용자 태그를 만들어 노트를 분류할 수 있습니다.

## 노트 만들기

노트 본문은 자유 텍스트입니다. 줄 바꿈(Enter)을 쓸 수 있어 한 노트에 여러 문단을 넣을 수 있습니다.

URL을 넣으면 저장 시 자동으로 링크로 바뀌어 클릭해 열 수 있습니다.

노트에는 연결된 구절을 원하는 만큼 붙일 수 있습니다. 구절 없이 텍스트만 있어도 됩니다.

텍스트 없이 구절만 있는 노트도 만들 수 있어, 나중에 다시 볼 구절을 표시해 두기에 좋습니다.

태그는 선택 사항이며 여러 태그를 붙일 수 있습니다.

## 태그로 필터

노트 페이지 위쪽 \"태그 필터\"로 태그를 고르면 해당 태그가 붙은 노트만 볼 수 있습니다.

태그 일치 방식은 세 가지입니다:

* **하나라도** — 선택한 태그 중 하나라도 있으면 포함. 다른 태그가 더 있어도 됩니다.
* **모두** — 선택한 태그를 **모두** 가진 노트만 포함. 하나라도 빠지면 제외. 그 외 태그는 상관없습니다.
* **정확히** — 선택한 태그 집합과 **딱 맞는** 노트만 포함. 하나라도 빠지거나 더 있으면 제외.

## 만든 날짜로 정렬

\"정렬\"로 다음 중 하나를 고릅니다:

* 최신순
* 오래된순

둘 다 노트를 **처음 만든** 시각을 기준으로 합니다. 수정해도 이 순서는 바뀌지 않습니다.

## 태그로 노트 정리

<div class="phone-frame">
  <img alt="My Bible Log 태그 페이지 스크린샷" src="/screenshots/sc11-note-tags.webp" />
</div>

노트 페이지의 \"태그\" 버튼으로 태그 페이지에 들어갑니다.

태그마다 색을 지정해 한눈에 구분할 수 있습니다.

설명을 적어 두면 태그 용도를 기록해 둘 수 있고, 설명 안의 URL도 링크로 바뀝니다.

태그마다 사용 중인 노트 개수가 보이며, 누르면 그 태그로 필터된 노트 페이지로 이동합니다.

아직 노트에서 쓰는 태그는 삭제할 수 없습니다. 삭제하려면 먼저 해당 태그를 떼거나 노트를 지우세요.
