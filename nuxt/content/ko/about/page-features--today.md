# 오늘 페이지

<div class="phone-frame">
  <img alt="My Bible Log 오늘 페이지 스크린샷" src="/screenshots/sc4-daily-goal.webp" />
</div>

오늘 페이지에서는 하루 성경 읽기 진행을 확인합니다. 위쪽 진행 막대는 오늘 일일 목표에 얼마나 가까운지 보여 줍니다. 예를 들어 목표가 86절이면 오늘 새로 86절을 읽으면 막대가 가득 찹니다.

오늘 읽기 기록도 여기서 보고 관리할 수 있습니다. 여러 번에 나누어 읽었다면 기록을 언제든 수정·삭제할 수 있습니다.

진행 막대 색 의미:

* **진한 파랑** — 오늘 새로 읽는 구절
* **어두운 파랑** — 예전에 읽은 구절(기준 시작일 이후)
* **반짝임** — 오늘 새 구절로 일일 목표를 달성했을 때

## 읽기 제안

다음에 무엇을 읽을지 막힐 때 오늘 페이지의 맞춤 제안을 활용하세요:

* 새 구절 찾기
* 일일 목표 채우기
* 성경 여러 부분 탐색

제안에는 다음이 섞여 있습니다:

* 신약
* 시편·지혜 문학
* 구약 기타

제안한 장을 읽어 가면 새 제안이 이어져 읽기가 지루해지지 않습니다.
