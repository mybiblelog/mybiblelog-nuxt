# 달력 페이지

<div class="phone-frame">
  <img alt="My Bible Log 달력 페이지 스크린샷" src="/screenshots/sc9-calendar.webp" />
</div>

달력 페이지에서는 시간에 따른 성경 읽기를 달력으로 봅니다.

날짜를 누르면 그날의 기록을 보거나, 필요하면 그날짜에 기록을 추가할 수 있습니다.

각 날짜에는 오늘 페이지와 비슷한 진행 막대가 있습니다. 진한 파랑과 어두운 파랑 두 가지로 채워집니다:

* 진한 파랑 — 그날 읽은 **새** 구절(기준 시작일 이후 기준).
* 어두운 파랑 — 그날 읽은 구절 전체. 모두 새 구절이면 어두운 파랑은 진한 파랑 뒤에 가려집니다.

막대는 하루 읽을 절 수 목표에 맞춰 채워집니다. 목표가 86절인데 새 구절을 43절 읽었다면 진한 파랑이 절반까지 찹니다.

일일 목표를 **읽은 구절 수**로 달성하면 그날 어두운 파랑 별이 나타나고, **새 구절**로 목표를 달성하면 금색 별이 나타납니다.
