# 기여하기

My Bible Log에 관심을 가져 주셔서 감사합니다. 도울 수 있는 방법은 많고, 작은 기여도 큰 의미가 있습니다. 잠깐이든 오래든 시간을 내어 주실 수 있습니다.

## My Bible Log 공유하기

가장 쉬운 기여는 다른 사람에게 알리는 것입니다. 입소문은 무료 성경 읽기 도구를 더 많은 사람에게 전하는 데 도움이 됩니다.

<content-section-grid>

### 💬 주변에 알리기

My Bible Log가 도움이 된다면 친구, 가족, 공동체에 추천해 보세요. 개인 추천이 새 도구를 알리는 가장 효과적인 방법인 경우가 많습니다.

### 📱 소셜 미디어에 공유하기

자주 쓰는 SNS에 My Bible Log를 공유해 보세요. 사용 후기나 추천 글 하나만으로도 많은 도움이 됩니다.

### 🔗 블로그·웹사이트에 링크

블로그나 사이트가 있다면 My Bible Log로 링크를 걸어 주세요. 더 많은 사람이 도구를 발견하고 프로젝트의 온라인 노출에도 도움이 됩니다.

### ⭐ GitHub에서 스타

GitHub 계정이 있다면 [My Bible Log 저장소](https://github.com/mybiblelog/mybiblelog-nuxt)에 스타를 눌러 주세요. 프로젝트 가시성과 가치를 보여 주는 데 도움이 됩니다.

</content-section-grid>

## 재정 후원

My Bible Log는 무료지만 운영에는 비용이 듭니다. 후원은 다음에 쓰입니다:

- **웹 서버** 호스팅
- **데이터베이스** 호스팅·유지
- 코드 편집기·AI 도우미 등 **개발 도구**
- 계정 인증·알림용 **이메일 서비스**

<content-section-grid>

### 💰 후원 방법

[Ko-fi 페이지](https://ko-fi.com/mybiblelog)를 통해 후원할 수 있습니다. 금액에 관계없이 감사하며, 모두 My Bible Log 운영·개선에 쓰입니다.

후원은 선택 사항이며, 후원 여부와 관계없이 My Bible Log는 항상 무료로 이용할 수 있습니다.

</content-section-grid>

## 기능 아이디어·피드백

아이디어와 피드백이 My Bible Log의 방향을 만듭니다.

<content-section-grid>

### 📝 피드백 보내기

기능 제안, 버그 신고, 일반 의견은 [피드백 양식](/feedback)을 이용해 주세요. 기능 아이디어에는 다음을 적어 주시면 좋습니다:

- 해결하고 싶은 문제
- 누가 혜택을 받는지
- 어떻게 동작하면 좋을지
- 예시·목업(있다면)

자세한 제안은 [GitHub](https://github.com/mybiblelog/mybiblelog-nuxt)에서 기능 요청 이슈를 열 수도 있습니다.

</content-section-grid>

## 번역 돕기

My Bible Log는 여러 언어를 지원하며, 번역을 넓히고 다듬는 데 도움이 필요합니다.

<content-section-grid>

### 🌍 지원 언어

현재 다음 언어를 지원합니다:

- 영어
- 스페인어
- 프랑스어
- 독일어
- 포르투갈어
- 우크라이나어
- 한국어

### 🌐 번역 기여 방법

위 언어에 능통하거나 아직 없는 언어를 추가하고 싶다면:

- 기존 번역 검토·개선
- 새 기능·콘텐츠 번역
- 새 언어 추가

코드베이스에서 번역이 어떻게 동작하는지는 GitHub의 [기여 가이드](https://github.com/mybiblelog/mybiblelog-nuxt/blob/main/CONTRIBUTING.md#internationalization)를 참고하세요.

시작하려면:

- GitHub에 이슈를 열어 논의
- 개선된 번역이 담긴 풀 리퀘스트 제출
- [피드백 양식](/feedback)으로 참여 의사 전달

</content-section-grid>

## 코드 기여

My Bible Log는 오픈 소스이며 코드 기여를 환영합니다.

<content-section-grid>

### 🚀 시작하기

1. [GitHub](https://github.com/mybiblelog/mybiblelog-nuxt)에서 **저장소 포크**
2. 설정·코딩 규칙은 **[기여 가이드](https://github.com/mybiblelog/mybiblelog-nuxt/blob/main/CONTRIBUTING.md)** 참고
3. 이슈를 고르거나 새 기능 제안
4. 변경 사항을 **풀 리퀘스트**로 제출

### ⚙️ 개발 환경

- **Node.js**와 TypeScript
- 프론트엔드 **Nuxt.js**
- API **Express**
- 데이터베이스 **MongoDB**

자세한 설정은 [기여 가이드의 개발 설정](https://github.com/mybiblelog/mybiblelog-nuxt/blob/main/CONTRIBUTING.md#development-setup)을 보세요.

</content-section-grid>

## 질문이 있으신가요?

기여나 아이디어에 대해 질문이 있으면:

- [피드백 양식](/feedback)으로 문의
- [GitHub](https://github.com/mybiblelog/mybiblelog-nuxt)에 이슈 열기
- [기여 가이드](https://github.com/mybiblelog/mybiblelog-nuxt/blob/main/CONTRIBUTING.md)에서 기술 내용 확인

My Bible Log에 관심을 가져 주셔서 감사합니다. 작은 기여도 모두에게 더 나은 도구를 만듭니다. 🙏
