# Contributing

Thank you for your interest in contributing to My Bible Log! There are many ways to help, and every contribution makes a difference. Whether you have a few seconds or several hours, there's a way for you to contribute.

## Share My Bible Log

The easiest way to contribute is to share My Bible Log with others. Word of mouth helps more people discover this free tool for tracking their Bible reading.

<content-section-grid>

### üí¨ Tell Someone About It

If you find My Bible Log helpful, consider telling a friend, family member, or someone in your community about it. Personal recommendations are often the most effective way for people to discover new tools.

### üì± Share on Social Media

Share My Bible Log on your favorite social media platforms. Whether it's a quick post about how you're using it or a recommendation to your followers, every share helps spread the word.

### üîó Add a Link from Your Blog or Website

If you have a blog or website, consider adding a link to My Bible Log. This helps others discover the tool and supports the project's visibility online.

### ‚≠ê Star the Project on GitHub

If you have a GitHub account, adding a star to the [My Bible Log repository](https://github.com/mybiblelog/mybiblelog-nuxt) is a great way to show your support! Stars help the project gain visibility and show that people find it valuable.

</content-section-grid>

## Financial Contributions

My Bible Log is free to use, but running it costs money. Financial contributions help cover:

- **Web server hosting** costs
- **Database** hosting and maintenance
- **Development tools** like code editors and AI assistants
- **Email services** for account verification and reminders

<content-section-grid>

### üí∞ How to Contribute Financially

You can now support My Bible Log financially through [its Ko-fi page](https://ko-fi.com/mybiblelog). Any amount helps, and all contributions go toward keeping My Bible Log running and improving.

Financial contributions are completely optional ‚Äî My Bible Log will always remain free to use regardless of whether you contribute financially.

</content-section-grid>

## Submit Feature Ideas and Feedback

Your ideas and feedback help shape the future of My Bible Log. We want to hear what features would be most helpful to you.

<content-section-grid>

### üìù How to Submit Feedback

You can submit feature requests, bug reports, or general feedback using [our feedback form](/feedback). When submitting a feature idea, please include:

- What problem the feature would solve
- Who would benefit from this feature
- How you envision the feature working
- Any examples or mockups (if you have them)

For more detailed feature requests, you can also create an issue on [GitHub](https://github.com/mybiblelog/mybiblelog-nuxt) using the feature request template.

</content-section-grid>

## Help with Translations

My Bible Log supports multiple languages, and we're always looking to expand language support or improve existing translations.

<content-section-grid>

### üåç Current Languages

My Bible Log currently supports:

- English
- Spanish
- French
- German
- Portuguese
- Ukrainian

### üåê How to Contribute Translations

If you're fluent in one of these languages (or a language we don't yet support), you can help by:

- Reviewing and improving existing translations
- Translating new features and content
- Adding support for a new language

For technical details on how translations work in the codebase, see the [Contributing Guide](https://github.com/mybiblelog/mybiblelog-nuxt/blob/main/CONTRIBUTING.md#internationalization) on GitHub.

To get started with translation contributions, you can:

- Open an issue on GitHub to discuss the translation work
- Submit a pull request with translation improvements
- Contact us through the [feedback form](/feedback) to express interest

</content-section-grid>

## Contribute Code

My Bible Log is open source, and we welcome code contributions! Whether you're fixing bugs, adding features, or improving documentation, your code contributions help make My Bible Log better for everyone.

<content-section-grid>

### üöÄ Getting Started

1. **Fork the repository** on [GitHub](https://github.com/mybiblelog/mybiblelog-nuxt)
2. **Read the [Contributing Guide](https://github.com/mybiblelog/mybiblelog-nuxt/blob/main/CONTRIBUTING.md)** for setup instructions and coding standards
3. **Pick an issue** to work on, or propose a new feature
4. **Submit a pull request** with your changes

### ‚öôÔ∏è Development Setup

The project uses:

For detailed setup instructions, see the [Contributing Guide](https://github.com/mybiblelog/mybiblelog-nuxt/blob/main/CONTRIBUTING.md#development-setup).

</content-section-grid>
